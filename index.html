Dino-Score!


For each game, four dinosaurs and a random number are shown below.

Clicking each dinosaur adds a certain number of points to your Dino-Score.

However, each dinosaur's value is unknown until you click it.

— You WIN if your Dino-Score EQUALS the random number. —

— You LOSE if your Dino-Score EXCEEDS the random number. —

For each new game, each dinosaur's value randomnly changes. GOOD LUCK!
